

**Description:** A custom Python replication of the Linux `ls` command. This foundational DevSecOps script takes a directory path from the terminal and categorizes its contents into files and folders.

**Core Logic:**

- Utilizes `sys.argv` to capture the directory path provided by the user in the terminal.
    
- Implements error handling: If no argument is provided, the script automatically defaults to scanning the current working directory.
    
- Uses `pathlib.Path.iterdir()` to loop through the directory contents.
    
- Routes items into specific data structures (lists) based on whether they are directories or files, and prints the formatted output.
    

**Engineering Learnings:**

- **Argument Parsing:** `sys.argv` is the standard method for handling single, straightforward terminal arguments.
    
- **Pathlib Mastery:** The `Path` object (capitalized) from the `pathlib` library is the modern way to handle filesystem routing in Python.
    
- **Data Structuring:** Storing data in lists or dictionaries before printing it makes the data manageable, reusable, and mathematically sortable.
